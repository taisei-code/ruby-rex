##

- 0x 16進数を表す

- String#hex
文字列を16進数で解釈して、整数を返す。
解釈できないときは、0を返す。

- 0b 2進数を返す
0b10が正しい定義

- String#oct
文字列を8進数で解釈して、整数を返す。
100が正しい定義
